# Smart-Renewable-Energy-Management-System
An Arduino-based smart monitoring system for renewable energy sources, fully simulated and verified in Proteus Design Suite.
An advanced, microcontroller-based smart monitoring and management system designed for renewable energy sources. This project provides real-time data tracking, automated load balancing, and high-fidelity hardware simulation.

---

## 🚀 Key Features
- **Real-Time Telemetry:** Continuous tracking of voltage, current, and power generated from renewable sources.
- **Automated Control Logic:** Smart load management based on dynamic environmental inputs and battery thresholds.
- **Embedded Architecture:** Highly optimized and modular code structured for standalone microcontroller deployment.
- **Verified Simulation:** Fully tested and benchmarked circuit design with zero-error simulation stability.

## 🛠️ Technology Stack & Tools
- **Hardware Simulation:** Proteus Design Suite
- **Development Environment:** Arduino IDE
- **Core Microcontroller:** Arduino / ATmega Series
- **Programming Language:** Embedded C / C++

## 💻 How to Setup and Run (Pre-compiled Hex Included)

Since the production-ready pre-compiled `.hex` binary file is already included in this repository, you do not need to compile the code manually in the Arduino IDE. Follow these simple steps to run the simulation instantly:

1. **Download the Files:** Download or clone this repository to your computer.
2. **Open the Simulation:** Launch **Proteus Design Suite** and open the circuit simulation file (`.pdsprj`) from the `simulation/` directory.
3. **Load the Hex File:** Double-click on the Arduino microcontroller board within the Proteus schematic workspace.
4. **Link the Binary:** In the properties window, click on the folder icon next to the **Program File** field, and select the pre-compiled `.hex` file provided in this repository.
5. **Execute Simulation:** Click the **Play / Run** button at the bottom-left corner of Proteus to watch the system operate in real-time.
